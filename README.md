# Pystub
Pystub is basic DNS stub server written in python with support for DNS over TLS.

# Upstream transports
- UDP
- TCP
- TLS

# Record Support
Support for the following record types has been implemented with more to come soon
- A
- NS
- CNAME
- SOA
- PTR
- MX
- TXT
- AAAA
- SRV
